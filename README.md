# 阿简成长实验室

这是「阿简成长实验室」的静态网站项目。

定位：一个普通人利用 AI、数据和长期复盘改变自己的成长记录。网站会长期更新，逐步沉淀为个人成长系统（Growth OS）。

## 页面结构

- `index.html`：首页 / Growth Dashboard
- `projects.html`：我的项目列表
- `archive.html`：成长档案时间轴
- `experiments.html`：实验库分类
- `whitepapers.html`：成长白皮书列表
- `handoffs.html`：项目交接卡列表
- `contact.html`：联系我

## 内容目录

- `projects/`：项目详情
- `archive/`：成长档案详情
- `experiments/`：实验分类详情
- `whitepapers/`：成长白皮书详情
- `handoffs/`：项目交接卡详情

## 后续新增内容

新增一篇成长白皮书：

1. 在 `whitepapers/` 下新增一个详情页。
2. 在 `whitepapers.html` 的列表中新增一条卡片链接。
3. 如有需要，同步更新首页 Growth Dashboard 数量。

新增项目、实验、交接卡同理。

## 部署

该项目使用 GitHub Pages 部署。后续准备绑定域名：阿巴阿巴.top。
